Descripción del Proyecto

Este proyecto final tiene como objetivo la instalación, configuración y gestión de distribuciones Linux, Docker, y Kubernetes en un entorno de trabajo real. Además, se crean aplicaciones, scripts y contenedores que se integran para crear un entorno funcional.

Integrantes del Grupo: Juan, Pedro, María
